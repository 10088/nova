==============
nova-conductor
==============

.. program:: nova-conductor

Synopsis
========

::

  nova-conductor [<options>...]

Description
===========

:program:`nova-conductor` is a server daemon that serves the Nova Conductor
service, which provides coordination and database query support for nova.
